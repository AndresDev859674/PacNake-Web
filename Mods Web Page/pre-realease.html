How to Install PacSnake Windows Pre-release

Place the file "PacSnake_4.0.1.0_pre-release.PNM" into the "Mods" folder located within the "PacSnake" directory. Facing errors? Here's the solution: Check if an "Assets" folder exists within the mod folder or inside the zip file. If it does, cut the "Assets" folder, navigate to the "PacSnake" directory, and paste it there. When prompted, confirm by selecting "Replace All."
