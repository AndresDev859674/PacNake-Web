PacSnake Mod Loader SDK

Make PacSnake run Mods

Description

The SDK allows Mods Loaders to be launched for PacSnake without requiring a PacSnake update. This enables faster deployment of new Mods Loaders without them being pre-installed in a PacSnake build.

This is required to some mods before a new build of PacSnake is realeased
