An Important Note

Sunday, March 29, 2026

I've lost interest in developing the game. I'm not popular, and I'm not going to continue developing it. What's the point of continuing a game without POPULARITY? That's why I haven't been able to update PacSnake. You could say I'm just taking a long time because I'm lazy. NO, I've lost so much interest that I couldn't even update the game, and I want to move on to other projects that are doing well. I'll also have to rewrite the code because it's unstable and a bit difficult to maintain. I can't continue with this mess. I don't want to say goodbye to this game; I want to release a few versions and keep going, but I'm having problems with my Linux system and the project, which won't even let me run the game to start programming because of Venv. See you in the future for the final update. Or not? Maybe there will be more updates. I promise some bug fixes for now. Perhaps I'll release an open-source version, I suppose, so they can improve this code with a bit of AI. I'm also focused on learning from Rust, and we'll see if I can release a complete version with a more or less reorganized codebase. I also hope I can continue with this code and get into the hell of rewriting it.

I'm afraid to share this on social media; I don't think Pacsnake is ready to share it publicly, even though it's already public.
