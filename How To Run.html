How to run an old PacNake (pip install ...)

    Hey! Keep in mind that if the game doesn't work, you didn't read it here.

  1. Install Python and download necessary libraries:
    • pip install pypresence - To run Discord RPC
    • pip install pygame - The Python Game Engine
    • pip install pygame_menu - An Engine for creating a main menu
    • pip install psutil - For system and process utilities
  2. Open PacNake and enjoy the game!
